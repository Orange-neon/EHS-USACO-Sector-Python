# Problem 2: Two Sum

## Description
Given an array of integers and a target sum, find the indices of two values in the array that add up to the target sum.

## Input Format
- First line: integer N (number of elements)
- Second line: N space-separated integers

## Output Format
- Single line: Two space-seperated integers (indices start with 0)

## Example
**Input:**

5 9

1 2 3 4 5

**Output:**
3 4
